# BiaMeet (بیا میت)

بیا میت یک سیستم ساده و کارآمد برای زمان‌بندی جلسات و رای‌گیری است.

## مرور کلی

بیا میت به کاربران اجازه می‌دهد جلساتی با چندین بازه زمانی ایجاد کنند و دیگران را برای رای‌گیری در مورد زمان‌های مورد نظر خود دعوت کنند.

## ویژگی‌ها

- **انواع جلسات منعطف**:
  - **زمان‌های مشخص**: پیشنهاد زمان‌های دقیق برای رای‌گیری.
  - **الگوی هفتگی**: تعیین روزهای مجاز و بازه زمانی برای جلسات تکرار شونده.
  - **بازه زمانی**: تعیین یک تاریخ و بازه زمانی کلی برای پیشنهاد زمان توسط کاربران.
- **اشتراک‌گذاری هوشمند**: پشتیبانی از Web Share API و کپی لینک.
- **داشبورد مدیریت**: مشاهده آمار سیستم در `/admin`.
- **تجربه کاربری**:
  - اعلان‌های Toast زیبا.
  - طراحی واکنش‌گرا (Responsive) برای موبایل.
  - پشتیبانی کامل از تقویم شمسی (جلالی).
  - شفافیت در رای‌گیری (نمایش نام رای‌دهندگان).

## معماری

- **بک‌اند**: Go + Fiber
- **دیتابیس**: SQLite
- **فرانت‌اند**: Vanilla JS + Tailwind CSS
- **دیپلوی**: Docker Compose

## توسعه محلی

### پیش‌نیازها

- Go 1.21+
- Docker & Docker Compose

### اجرا

1. **کلون کردن مخزن**:

   ```bash
   git clone https://github.com/malekpouri/biameet.ir.git
   cd biameet.ir
   ```

2. **اجرا با داکر**:

   ```bash
   docker-compose up --build
   ```

   برنامه در آدرس `http://localhost:8085` در دسترس خواهد بود.

3. **اجرای دستی بک‌اند**:
   ```bash
   cd backend
   go run cmd/main.go
   ```
