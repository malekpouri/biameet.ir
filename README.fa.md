# BiaMeet (بیا میت)

بیا میت یک سیستم ساده و کارآمد برای زمان‌بندی جلسات و رای‌گیری است.

## مرور کلی

بیا میت به کاربران اجازه می‌دهد جلساتی با چندین بازه زمانی ایجاد کنند و دیگران را برای رای‌گیری در مورد زمان‌های مورد نظر خود دعوت کنند.

## ویژگی‌ها

- ایجاد جلسات با عنوان و نام ایجاد کننده.
- تعریف چندین بازه زمانی (UTC).
- اشتراک‌گذاری لینک برای رای‌گیری.
- رای‌گیری کاربران برای بازه‌های زمانی.
- پشتیبانی از تاریخ شمسی (جلالی) در رابط کاربری.

## معماری

- **بک‌اند**: Go + Fiber
- **دیتابیس**: SQLite
- **فرانت‌اند**: Static HTML/JS + Tailwind CSS
- **دیپلوی**: Docker Compose

## توسعه محلی

### پیش‌نیازها

- Go 1.21+
- Docker & Docker Compose

### اجرا

1. **کلون کردن مخزن**:

   ```bash
   git clone https://github.com/malekpouri/biameet.ir.git
   cd biameet.ir
   ```

2. **اجرا با داکر**:

   ```bash
   docker-compose up --build
   ```

3. **اجرای دستی بک‌اند**:
   ```bash
   cd backend
   go run cmd/main.go
   ```
